Suitable Crop Prediction using Machine Learning

This project aims to predict the most suitable crop for cultivation based on various soil parameters using supervised machine learning algorithms. The model takes into account features such as soil pH, nitrogen (N), phosphorus (P), potassium (K), organic matter, moisture content, and other geospatial attributes. Key highlights: Utilized algorithms like Random Forest, Decision Tree, and SVM for classification. Achieved high accuracy on curated and cleaned soil datasets. Performed feature engineering, data normalization, and hyperparameter tuning to optimize model performance. The application can support precision agriculture and help farmers make data-driven decisions. 🧪 Tech Stack: Python, scikit-learn, pandas, NumPy, matplotlib 📊 Use Case: Crop recommendation for sustainable farming practices

Sea Port View – OpenGL-Based 2D Graphics Simulation

Sea Port View is a computer graphics project that simulates a 2D animated scene of a sea port using OpenGL. The program includes realistic elements like ships, water waves, cranes, moving clouds, and the sun, offering a visually engaging representation of a coastal port environment. Key features: Designed using OpenGL (GLUT) for rendering dynamic 2D graphics. Implements translation and animation logic for moving ships, clouds, and other port components. Includes user interaction elements such as starting and stopping animation. Demonstrates core concepts of computer graphics: object transformation, rendering, timing, and double buffering. 🎯 Use Case: Academic graphics simulation project 🧪 Tech Stack: C++, OpenGL, GLUT.

DSRS (Dhaka Subway Reservation System)

Dhaka Subway Reservation System is a GUI-based application designed to simulate an efficient metro/subway ticket reservation system tailored for Dhaka’s urban transport needs. The system allows users to view routes, check available seats, and book or cancel tickets in real-time. 🔧 Core Features: 🛤 Station Selection: Choose source and destination stations from a dropdown or list. 🎫 Ticket Booking: Book one or more seats per user with real-time availability check. 🧾 Reservation Details: Generate reservation summary with passenger info, fare, and route. ❌ Ticket Cancellation: Cancel reserved tickets using ticket/reference ID. 📅 Timetable View: Display next available trains and timing between selected stations.
